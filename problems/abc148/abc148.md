## 思考
2数の最小公倍数(LCM)を求める問題。
num crateに関数があるが、gcdを実装してlcmを求めた。
```
use proconio::{input, marker::Chars};

pub fn main() {
    input! {
        a: i64,
        b: i64
    };
    println!("{}", (a*b) / gcd(a, b));
}

fn gcd(a: i64, b: i64) -> i64 {
    let r = a.max(b) % a.min(b);
    if r == 0 {
        return a.min(b);
    }
    return gcd(a.min(b), r);
}
```
