# C

## 思考

条件を満たす組合せを全部格納していく方法が考えられる。

```
let mut set = HashSet::<(i32, i32, i32)>::new();
for a in 1..n {
  for b in 1.. {
    if a*b >= n {
      break;
    }
    set.insert((a, b, n-a*b));
  }
  println!("{}", set.len());
}
```

これは O(N^2)なので間に合わない。
ということでループを緩和していくことを考える。
b の数え上げが簡単そうなので、直して AC。

```
use proconio::{input, marker::Chars};

pub fn main() {
    input! {
        n: i32
    };
    let mut count = 0;
    for a in 1..n {
        count += (n-1)/a;
    }
    println!("{}", count);
}
```
