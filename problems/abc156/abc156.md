# C
## 思考
数直線上の点がいくつか与えられるので、すべての点からの距離の総和が最小になる座標を答える問題。
与えられた点から離れれば離れるほど、距離が大きくなることは直感的に分かるので、
探索範囲は与えられる点の範囲だけでいいことがわかる。
全探索して最小値を答える。
```
use proconio::{input, marker::Chars};
type VS = Vec<String>;

pub fn main() {
    input!{
        n: usize,
        xn: [i32; n]
    };
    let mut min = std::i32::MAX;
    for p in 0..=100 {
        let cost = xn.iter().map(|x| (x-p).pow(2)).sum();
        if cost < min {
            min = cost;
        }
    }
    println!("{}", min);
}
```
