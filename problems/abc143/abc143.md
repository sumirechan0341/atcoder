## 思考
文字列が与えられて、隣接する文字が等しいとき融合するので、最終的に何文字になるか求める問題。
先頭から2文字ずつ見ていくだけ。
```
use proconio::{input, marker::Chars};

pub fn main() {
    input! {
        n: usize,
        s: Chars
    };
    let mut count = 0;
    for i in 0..n-1 {
        if s[i] == s[i+1] {
            count += 1;
        }
    }
    println!("{}", n-count);
}
```
